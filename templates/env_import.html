How to export from the Govee app:
Open the Govee Home app → tap your H5179 device → tap the graph icon → select a date range → tap Export. The exported .csv file will have Timestamp, Temperature, and Humidity columns. Both °F and °C exports are supported — the unit is detected automatically from the column header.

Expected CSV Format

The importer expects a header row followed by data rows. Column order does not matter. Headers must contain the keywords Time/Date, Temp, and Humid. A CO2 or Carbon column is captured automatically if present.

Timestamp,Temperature(°F),Humidity(%RH),CO2(ppm)
2026-05-10 06:00:00,71.2,91.4,842
2026-05-10 06:10:00,71.5,90.8,856
2026-05-10 06:20:00,71.3,91.1,831

°C exports are converted to °F automatically. Percentage signs, degree symbols, and units in value cells are stripped during parsing. CO2 column is optional — temp/humidity-only exports still import cleanly.
