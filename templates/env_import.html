How to export from the Govee app:
Open the Govee Home app → tap your H5179 device → tap the graph icon → select a date range → tap Export. The exported .csv file will have Timestamp, Temperature, and Humidity columns. Both °F and °C exports are supported — the unit is detected automatically from the column header.
1 — Select Chamber
Readings will be linked to this chamber
2 — Choose File
.csv file exported from the Govee Home app
3 — Import
Rows with a timestamp already in the database for this chamber will be skipped automatically — re-importing the same file is safe.

Expected CSV Format

The importer expects a header row followed by data rows. Column order does not matter as long as the headers contain the keywords Time/Date, Temp, and Humid.

Timestamp,Temperature(°F),Humidity(%RH)
2026-05-10 06:00:00,71.2,91.4
2026-05-10 06:10:00,71.5,90.8
2026-05-10 06:20:00,71.3,91.1

°C exports are converted to °F automatically. Percentage signs and degree symbols in value cells are stripped during parsing.
