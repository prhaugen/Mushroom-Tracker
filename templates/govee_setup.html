How it works

The app polls each enabled sensor every 10 minutes and stores the reading in the environment log for its mapped chamber. A reading is skipped if one already exists within the last 8 minutes (prevents duplicates on quick manual syncs).

To backfill historical gaps before today, use Import Govee CSV — the Govee app can export any date range.

Shelf is optional — set it if you have multiple sensors at different heights so the Batch Detail charts can filter to the right sensor.
